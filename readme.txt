== Description ==

 Create a customize video player using elementor widget.

Major features include:

* Create a customize shape for your video player
* Available as elementor widget 
* Easy to customize features

== Installation ==

Upload the Awesome Video Player plugin to your website, activate it, go to elementor, find awsome video player widget, drag it to page and customize it using controls.

There you go!
